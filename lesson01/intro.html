jQuery.intro jQuery 简介

jquery文件整体由一个匿名函数自执行 
           (function( window, undefined ) {

            // ......

            window.jQuery = window.$ = jQuery; // line:8826
           })( window );

优点:

  • 函数体内都为局部变量,有效减少全局变量污染
  • 也可有效的压缩代码
  • 避免undefined,在一些浏览器上可以被赋值。js假值

在函数体内，通过全局windows的属性扩展成为全局变量。
