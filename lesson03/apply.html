apply()方法 和 call()方法

间接调用函数。call() 和 apply() 的第一个实参数 是调用上下文, 在函数体内通过 this 来获得引用。

  • 在ECMA5严格模式中,call() 和 apply() 的第一个参数会变为this 的值。(包括 null, undefined)
  • 在ECMA3 和非严格模式,传入null 和 undefined 都会被 全局对象替代，而其他原始值则会被相应的包装对象替代。 
         // call(), 的参数 除第一个实参的剩余参数
         f.call(obj, arg1, arg2, arg3);

         // apply() 第二个参数为一个数组，作为传入函数的参数
         // 实际上第二个也能接受 arguments数组
         f.apply(obj ,[arg1, arg2, arg3]);
